Blazy failing with prerender

If you see the logs in a browser such as Chrome or Firefox, you'll see the image being loaded, and the variable window.prerenderReady being set to true after the blazy has done its thing and the image has loaded.

However, with prerender this doesn't happen. Eventually the service times out and serves the page without the image being loaded, even though the window.prerenderReady has not been set to true yet.
